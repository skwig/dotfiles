# adw-gtk3 assets

Image assets vendored from [adw-gtk3](https://github.com/lassekongo83/adw-gtk3)
(`src/assets/`), used by the generated GTK3 light theme
(`quickshell/matugen/templates/gtk-light-colors.css`, itself derived from
adw-gtk3's `gtk.css`).
